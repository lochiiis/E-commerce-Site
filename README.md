# E-commerce Website

A simple full-stack e-commerce web application built with the MERN stack (MongoDB, Express, React, Node.js) and Redux for state management.

## 🔧 Tech Stack

- **Frontend**: React, Redux, Axios
- **Backend**: Node.js, Express, MongoDB
- **State Management**: Redux + Thunk

## 🚀 Features

- User registration & login (JWT)
- Product listing & details
- Add to cart & checkout
- Order placement
- Admin panel for managing products & orders

## 📦 Installation

```bash

# Backend setup
cd backend
npm install
npm run server

# Frontend setup
cd frontend
npm install
npm start
